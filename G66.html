Пресс для пюре 559 руб. /шт. 615 руб. - + 100 шт.

Пресс состоит из основания с отверстиями и длинной ручки с ушком для подвешивания.

Характеристики:

Длинная ручка пресса создает максимальное удобство использования: превращать продукт в пюре можно в самой глубокой посуде! Углубления с обеих сторон ручки помогают равномерно распределять прилагаемую силу по всей поверхности плоской части пресса. Продолговатые отверстия на плоской части пресса позволяют быстро и без особых усилий измельчать и разминать отварной картофель или вареные овощи. Благодаря закругленной форме ручка можно быть уверенным, что ваша рука никогда не соскользнет с нее при измельчении большого количества продуктов.

Использование:

Пресс для пюре можно использовать для приготовления пюре из разнообразных вареных овощей:моркови, зеленого горошка, тыквы, цветной капусты, очищенных томатов, цуккини. Прекрасно подходит для приготовления фруктово-ягодных пюре, супов-пюре, блюд для детского питания.
